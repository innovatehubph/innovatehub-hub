# Quick Templates

Pre-configured prompts for common feature requests. Click any template to populate the input field with a detailed prompt.

## CRM Contacts
> Create a CRM Contacts page that displays a table of customer contacts with name, email, phone, company, status, and last contact date. Include search, pagination, and the ability to add/edit contacts.

## Email Campaigns
> Create an Email Campaigns management page with a table showing campaign name, subject, recipients, send date, open rate, and click rate. Include campaign creation form.

## Analytics Dashboard
> Create an Analytics & Reports page showing business metrics. Include line charts for messages over time, bar charts for campaign performance, and summary stat cards.

## Payment Processing
> Create a Payment Processing page for PlataPay integration. Show stat cards for total transactions, revenue, pending payments, and failed transactions.

## Inventory Manager
> Create an Inventory Management page. Show stat cards for total SKUs, low stock items, and out of stock count. Include data table with product name, SKU, quantity, reorder level.

## Telegram Integration
> Create a Telegram Bot Integration page. Show configuration section for bot token, webhook URL, and command list. Include message log table.
