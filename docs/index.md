# InnovateHub Business Hub

A full-stack **Facebook Business Management Platform** with AI-powered development capabilities. Built on Back4App (Parse Server), React + TypeScript, and Claude AI.

![Dashboard](assets/screenshots/02-nav-dashboard.png)

## Key Features

- **Multi-Business Management** — Manage multiple Facebook Business accounts from a single dashboard
- **Messenger Integration** — View and respond to Facebook Messenger conversations
- **Ad Campaign Management** — Create, monitor, and optimize Facebook ad campaigns
- **Product Catalog** — Manage product listings synced with Facebook Shops
- **Bot Flow Builder** — Design automated conversation flows for Messenger bots
- **Webhook Processing** — Real-time event processing from Facebook's webhook API
- **AI Workshop** — Claude AI-powered code generation that builds new features automatically
- **Token Store** — Manage and refresh Facebook API tokens securely

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React 19 + TypeScript, Vite 7, Tailwind CSS v4 |
| Backend | Back4App (Parse Server), Cloud Code (Node.js) |
| Database | Back4App Data Store (MongoDB) |
| AI Engine | Claude AI via AI Proxy |
| Deployment | Back4App Hosting + VPS |
| Testing | Puppeteer E2E + Automated Screenshots |
