# Users & Roles

Manage dashboard users and their access levels.

![Users & Roles](../assets/screenshots/12-nav-users-roles.png)

## User Management

- **View users** — List all dashboard users
- **Add users** — Invite new team members
- **Edit roles** — Assign admin, editor, or viewer permissions
- **Deactivate** — Disable user access without deleting

![Users Data Table](../assets/screenshots/35-data-users-roles.png)

## Role Permissions

| Role | Dashboard | Data | Settings | AI Workshop |
|------|-----------|------|----------|-------------|
| Admin | Full | Full CRUD | Full | Full |
| Editor | View | Create/Edit | View | Generate |
| Viewer | View | Read only | None | None |
