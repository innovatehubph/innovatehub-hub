# Ads & Campaigns

Create, monitor, and optimize your Facebook advertising campaigns.

![Campaigns Page](../assets/screenshots/07-nav-ads-campaigns.png)

## Campaign Management

- **Create campaigns** — Set up new ad campaigns with targeting and budget
- **Monitor performance** — Real-time metrics: impressions, clicks, CTR, spend
- **Manage ad sets** — Configure audience targeting, placements, and scheduling
- **Track conversions** — Monitor leads, purchases, and custom conversions

## Campaign Fields

| Field | Description |
|-------|-------------|
| Name | Campaign name |
| Objective | Marketing objective (awareness, traffic, conversions) |
| Status | Active / Paused / Completed |
| Budget | Daily or lifetime budget |
| Spend | Amount spent to date |
| Results | Key metric based on objective |
