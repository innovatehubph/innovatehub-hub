# Messenger & Conversations

View and manage all Facebook Messenger conversations across your business pages.

![Messenger Page](../assets/screenshots/04-nav-messenger.png)

## Conversations List

Browse all conversation threads with:
- Customer name and profile
- Last message preview
- Timestamp
- Business page association
- Read/unread status

## Message History

Click any conversation to view the full message thread:
- Text messages
- Attachments (images, files)
- Quick replies and postbacks
- Bot-generated responses

## Data Table

![Messenger Data](../assets/screenshots/33-data-messenger.png)

The messenger data table supports:
- **Search** — Filter conversations by customer name or content
- **Sort** — Order by date, customer, or status
- **Pagination** — Navigate through large conversation lists
